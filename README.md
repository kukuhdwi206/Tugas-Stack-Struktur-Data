### Penjelasan Singkat Kode
Program ini mengimplementasikan konsep **Stack (Tumpukan)** secara manual menggunakan array, sesuai dengan aturan tugas (tanpa library Java util).

**Fitur Utama:**
1. **Push**: Menambahkan tiket ke posisi paling atas.
2. **Pop**: Mengambil/menghapus tiket yang terakhir masuk (LIFO).
3. **Hitung Total (Bonus)**: Menjumlahkan seluruh harga tiket yang ada di dalam tumpukan.
